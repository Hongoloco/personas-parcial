# Web de estudio - Personas

Web estatica para preparar el primer parcial de Personas / Derecho Civil.

## Abrir localmente

Abrir `index.html` en el navegador.

## Publicar online con GitHub Pages

Este repo incluye un workflow en `.github/workflows/pages.yml`.

1. Crear un repositorio en GitHub.
2. Agregarlo como remoto:

```powershell
git remote add origin https://github.com/TU_USUARIO/TU_REPO.git
git branch -M main
git push -u origin main
```

3. En GitHub, entrar a `Settings > Pages`.
4. En `Build and deployment`, elegir `GitHub Actions`.
5. La web queda disponible en la URL de GitHub Pages del repositorio.
